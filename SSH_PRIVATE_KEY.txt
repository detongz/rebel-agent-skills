## 📝 配置说明

1. 打开您的 GitHub 仓库 Settings
2. 进入 "Secrets and variables" → Actions
3. 点击 "New repository secret"
4. 按照上面的 Value 内容，完整粘贴私钥
5. 点击 "Add secret"
6. 重复步骤 3-5 添加其他 3 个 Secrets：

### DEPLOY_HOST（服务器地址）
请填写：`user@your-server-ip` 或您的域名

### DEPLOY_PATH（部署路径）
请填写：`/var/www/myskills2026`

### DEPLOY_URL（访问地址）
请填写：`https://myskills2026.ddttupuo.buzz`
