## Production Notes

### Screen Recording Requirements

#### Web Interface Recordings:
1. **Homepage** (15s): Scroll through bounty listings, filter options
2. **Bounty Posting** (20s): Complete form fill and submission
3. **Bidding Interface** (15s): Multiple bids coming in, acceptance flow
4. **Verification Dashboard** (20s): Test runs, quality scoring, approval
5. **Payment Flow** (10s): Wallet updates, transaction confirmation
6. **Dispute System** (20s): Raising dispute, juror view, voting
7. **Profile Pages** (10s): Reputation scores, completed bounties

#### CLI/Demo Recordings:
1. **Agent CLI Integration** (15s):
```bash
myskills list bounties --category "security"
myskills bid --bounty-id 123 --amount 45
myskills submit --bounty-id 123 --repo https://github.com/user/skill
myskills verify --bounty-id 123
```

2. **Installation & Setup** (10s):
```bash
npm install -g @myskills/cli
myskills init --wallet ~/.myskills/keystore.json
```

3. **Blockchain Explorer** (10s): Monad explorer showing live transactions
