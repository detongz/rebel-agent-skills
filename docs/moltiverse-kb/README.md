## üé¨ Demo Video Guidelines

### Requirements
- **Length**: 2 minutes maximum
- **Content**: Show core features in action
- **Quality**: Clear audio and visuals
- **Hosting**: Public link (YouTube, Vimeo, Loom, etc.)

### Suggested Structure
1. **0:00-0:20**: Introduction - What is OpenClaw Protocol?
2. **0:20-0:50**: Core features - Agent marketplace, tokenized skills
3. **0:50-1:20**: Live demo - Agent posting and executing bounty
4. **1:20-1:45**: Technical highlights - Monad integration, MCP protocol
5. **1:45-2:00**: Call to action - GitHub repo, future roadmap

### Recording Tools
- **OBS Studio** - Free, professional screen recording
- **Loom** - Easy recording and hosting
- **QuickTime** - Built-in Mac screen recording
- **PowerPoint/Keynote** - Slideshow with narration

## üêõ Common Issues & Solutions

### Demo Video
- **Too long?** Focus on core features, cut filler
- **Audio issues?** Use external mic or text-to-speech
- **Hosting problems?** Try YouTube (unlisted) or Vimeo

### Deployment
- **Frontend not building?** Check Node.js version (18+)
- **Contracts failing?** Verify Monad RPC endpoint
- **Links not public?** Check repo visibility and hosting

### Submission Form
- **Character limits?** Link to docs instead of pasting
- **Links not working?** Test all before submitting
- **Missing fields?** Use the checklist above
