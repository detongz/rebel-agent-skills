## 核心差异化

### 1. 安全第一
```
安装前必须扫描:
  ↓
免费基础扫描 (grep + npm audit)
  ↓
显示安全评分 0-100
  ↓
用户决定: 继续安装 / 放弃 / 深度扫描
```

### 2. 深度扫描（付费）
```
VirusTotal API 集成:
  • 文件哈希检查
  • 恶意软件数据库
  • 威胁情报分析
  • 行为分析
  • 沙箱执行

定价: $0.01 - $0.05 每次扫描
```

### 3. 无需登录
```
Vercel Skills:  ✅ 无需登录
ClawHub:       ❌ 需要 GitHub OAuth
MySkills:      ✅ 无需登录 (聚合两者)
```

## 技术实现

### 搜索命令
```typescript
// 并行调用两个来源
const [clawhubResults, vercelResults] = await Promise.all([
  exec('npx clawhub search security'),
  exec('npx skills find security')
]);
```

### 扫描命令
```typescript
// 1. 克隆仓库
git clone --depth 1 <repo> /tmp/scan

// 2. 代码分析
grep -r "eval|exec|child_process" /tmp/scan

// 3. 依赖检查
cd /tmp/scan && npm audit --json

// 4. 计算评分
score = (codeScore + depScore) / 2
```

### 安装命令
```typescript
// 根据来源调用对应命令
if (source === 'clawhub') {
  exec('npx clawhub install ' + skill);
} else if (source === 'vercel') {
  exec('npx skills add ' + repo);
}
```

3. **企业服务**
