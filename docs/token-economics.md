## 代币用途

### 1. 支付工具 (Payment)
- **用户打赏**：用 $MSKL 打赏 Skill 创作者
- **项目方赞助**：用 $MSKL 赞助特定类别的 Skills

### 2. 治理权 (Governance)
- **提案投票**：持有 $MSKL 可以对平台提案投票
- **决策权重**：1 $MSKL = 1 票

### 3. 广告位购买 (Sponsorship)
- **赞助竞价**：项目方通过消耗 $MSKL 竞价赞助位置
- **类别独家**：可以购买某个 Skill 类别的独家赞助权

### 4. 创作者挖矿 (Mining)
- **打赏挖矿**：创作者收到打赏时，额外获得 $MSKL 奖励
- **热度挖矿**：Skill 安装量/使用量越高，获得更多 $MSKL

---

## 挖矿机制

### 创作者挖矿公式

```
每日挖矿奖励 = 基础奖励 × 热度系数 × 质量系数

其中：
- 基础奖励：每日固定 100,000 $MSKL
- 热度系数：min(1 + 安装量/1000, 10)
- 质量系数：1 + 好评率 × 0.5
```

### 举例

| Skill | 安装量 | 好评率 | 热度系数 | 质量系数 | 日收益 |
|-------|-------|--------|----------|----------|--------|
| A | 100 | 80% | 1.1 | 1.4 | 154 $MSKL |
| B | 5,000 | 95% | 6 | 1.475 | 885 $MSKL |

## 销毁机制

### 每次打赏销毁 2%

当用户用 $MSKL 打赏创作者时：
- 98% 给到创作者钱包
- 2% 自动销毁（通缩）

### 预期销毁速率

假设每日打赏量 100万 $MSKL：
- 每日销毁：20,000 $MSKL
- 年度销毁：7,300,000 $MSKL (约 0.73% 总量)

## 经济飞轮

```
┌─────────────────────────────────────────────────────────────┐
│                        经济飞轮                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   更多 Skills        更高热度        更多打赏        代币升值 │
│   ──────────►      ─────────►      ──────────►      ──────► │
│   创作者入驻        用户参与        生态繁荣        吸引投资 │
│      ▲                                                            │
│      │                                                            │
│      └──────────────────  代币激励  ───────────────────────────┘ │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**飞轮逻辑**：
1. 代币激励吸引创作者上传 Skills
2. Skills 越多，用户越愿意使用平台
3. 用户使用越多，打赏越多
4. 打赏越多，代币需求越大，价格上涨
5. 代币上涨，创作者更有动力创作
6. 回到步骤 1，循环放大

## 风险控制

### 防刷量机制

1. **地址限制**：同一地址 24 小时内只能打赏 10 次
2. **金额限制**：单次打赏最低 10 $MSKL，最高 10,000 $MSKL
3. **黑名单**：检测异常行为（如自刷）可冻结账户

### 价格稳定机制

1. **金库干预**：当价格波动超过 20%，金库可买入/卖出稳定价格
2. **打赏锁仓**：大额打赏（>1000 $MSKL）分 7 天线性释放
